# Astrid Eichhorn

Astrid Eichhorn is a physicist at Heidelberg University.

## Perspectives

Eichhorn raised a foundational challenge to the [bootstrap program](../concepts/bootstrap-method.md)'s application to [string theory](../theories/string-theory.md). She suggested that the ultraviolet regime — the extreme high-energy domain where a [UV completion](../concepts/uv-completion.md) is needed — might involve spacetime configurations so extremely curved that flat-space [scattering amplitude](../concepts/scattering-amplitudes.md) descriptions become meaningless.

If correct, this would undermine the framework on which current bootstrap derivations of string theory rely, since those derivations assume scattering in flat spacetime.
