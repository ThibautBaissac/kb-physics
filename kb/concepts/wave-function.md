# Wave Function

The wave function is the central mathematical object of quantum mechanics, introduced by Erwin Schrödinger in 1926. It represents the state of a quantum system as a vector in a [Hilbert space](../concepts/hilbert-space.md), encoding the probabilities of all possible outcomes of a measurement as a superposition.

## The Superposition Problem

Before measurement, all possible values of a property (e.g., an electron's position) coexist in the wave function as a superposition — each potentially observable with some probability. Measurement seemingly causes the superposition to "collapse" into a single definite outcome. This is the heart of the [measurement problem](../open-questions/measurement-problem.md).

The wave function cannot tell us what a quantum system is like before measurement. It tells us only what we'll see *if we look*. As Jeffrey Bub (University of Maryland) puts it, quantum uncertainty is "a new sort of ignorance about something that doesn't yet have a truth value."

## Interpretive Debate: Epistemic vs. Ontic

The nature of the wave function is the central question dividing quantum interpretations:

- **Epistemic** (Copenhagen): The wave function describes our knowledge of the quantum world, not reality itself. Collapse reflects our updated knowledge.
- **Ontic** (Many-Worlds): The wave function *is* reality — all branches exist simultaneously in parallel universes. We experience only one branch.
- **Epiontic** (Zurek): [Wojciech Zurek](../people/wojciech-zurek.md) argues the wave function is both. Before [decoherence](../concepts/decoherence.md), all possibilities exist (ontic). Decoherence and [quantum Darwinism](../concepts/quantum-darwinism.md) select one as observable (epistemic). The unselected possibilities never grow into observable realities.
- **[QBism](../concepts/qbism.md)** ([Chris Fuchs](../people/chris-fuchs.md)): The wave function is entirely first-person — a catalog of a specific agent's degrees of belief about the consequences of their actions. It says nothing about the world independent of that agent.
- **[Relational](../concepts/relational-quantum-mechanics.md)** ([Carlo Rovelli](../people/carlo-rovelli.md)): The wave function is always defined relative to another physical system (the observer). No absolute wave function of the universe exists; all descriptions are relational.

## Heisenberg's Cut

Heisenberg proposed a "cut" dividing the quantum and classical worlds. Below the cut, quantum mechanics applies; above it, classical physics reigns. But modern experiments show quantum behavior in objects visible under optical microscopes, blurring any sharp boundary. Decoherence theory provides a mechanism for the transition without requiring an arbitrary cut.

## Complex-Valued Nature

The wave function is fundamentally [complex-valued](complex-numbers-in-quantum-mechanics.md) — Schrödinger's equation features the imaginary number *i* (√−1), and the wave function lives in a complex [Hilbert space](hilbert-space.md). Although measurements always return real values, the complex structure governs how quantum states evolve and interfere. In 2025, several groups showed that quantum mechanics can be reformulated using only real numbers, but the resulting theories retain the hallmarks of complex arithmetic — suggesting something deeply natural about the complex formulation.
