# Bootstrap Method

The bootstrap is a methodology in theoretical physics that starts with fundamental physical principles — such as symmetry, unitarity, and causality — and derives what theories necessarily follow, rather than postulating a theory and checking its consequences.

## Application to String Theory

Recent bootstrap work has reframed the [string theory](../theories/string-theory.md) debate. Instead of asking whether string theory is correct, researchers ask whether its foundational assumptions are reasonable — and whether string theory is the *unique* theory satisfying those assumptions.

[Cliff Cheung](../people/cliff-cheung.md) described the reversal: "People have studied for decades what string theory implies, and we're asking, 'What implies string theory?'"

### Key Results

- **"Strings From Almost Nothing" (August 2025):** Cheung and collaborators showed that assuming [ultrasoftness](../concepts/ultrasoftness.md) uniquely produces string theory's [scattering amplitudes](../concepts/scattering-amplitudes.md).
- **"String Theory From Maximal Supersymmetry" (January 2026):** [Henriette Elvang](../people/henriette-elvang.md) demonstrated that N=4 [supersymmetry](../concepts/supersymmetry.md) in quantum field theory uniquely requires string theory as its UV completion.

### Limitations

[Pedro Vieira](../people/pedro-vieira.md) cautioned that Elvang's result currently holds only for tree-level amplitudes — the full quantum (loop-level) picture remains open. Critics like Peter Woit dismissed the ultrasoftness result as circular, since that property was already known to characterize string theory.
